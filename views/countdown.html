The Objective

Stop the detonation. You are dropped into a city with zero intel. Use logical reasoning to connect environmental clues before the clock hits zero.

The Mechanic

No hand-holding. The game uses a non-linear interaction system where the environment itself tells the story. Investigation is the only path to the target.

Mission Parameters

Infiltration

Exploration & Movement

Third-person WASD movement with a dynamic camera system optimized for tight urban exploration.

Intelligence

Interaction System

C#-driven raycasting system allows the player to examine objects and collect narrative fragments.

Execution

Timed Clue Progression

A sequential logic gate system: Clues must be interpreted correctly to reveal the final defusal point.

System Architecture

  • Engine: Unity 3D
  • Logic: C# Programming
  • Assets: Blender (3D Modeling)
  • Platform: PC / Windows

Atmospheric Design

Low-key lighting and spatial audio design are used to create a "Realistic Thriller" experience without relying on supernatural tropes.
